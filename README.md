# Pad-and-Align-Python-String

## Input: 
  A list of strings containing various letters of different length
  Ex: l = ["washington", "test", "key", "random"]
  
## Output:
  To align based on the length of the longest element
  Ex: w|a|s|h|i|n|g|t|o|n
      *|*|*|*|*|*|t|e|s|t
      *|*|*|*|*|*|*|k|e|y
      *|*|*|*|r|a|n|d|o|m
